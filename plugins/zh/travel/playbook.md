# 旅行打包插件

## 这个插件做什么

帮助用户规划旅行打包。维护一个主打包模板，根据行程天数、天气和出行人员自动缩放。生成行程专属打包清单并发送出行前提醒。

## 安装

安装此插件时，在用户的 `Preferences.md` 的"已安装插件"部分添加以下内容：

```
- travel：旅行打包和行程规划。触发词："添加行程"、"要带什么"、"生成打包清单"、"我的行程"、"[行程]需要带什么"、"要出去旅行"。→ 按 travel 插件 playbook 操作。数据：plugins/travel/my_packing_template.md, plugins/travel/my_trips.md
```

## 主模板

助手维护一个 `my_packing_template.md`，包含用户的个性化打包物品。模板按类别组织，基准数量按 3 天行程计算。

### 默认类别

**证件和必需品：** 护照、钱包、保险卡、药品
**电子设备：** 手机、电脑、充电宝、充电线、充电器
**衣物 — 成人：** 上衣、裤子、内衣、袜子、睡衣（按天数缩放）
**衣物 — 儿童：** 同上类别，儿童需要额外备用
**洗漱用品：** 牙膏、牙刷、护肤品、防晒霜
**食物和饮料：** 水、零食、特殊饮食需求
**宠物：** 如适用 — 狗粮、碗、牵引绳、药品

### 缩放规则

**衣物数量 = 基准 + ceil((行程天数 - 3) / 2)**
- 3天：基准数量
- 5天：每项+1
- 7天：每项+2
- 10天：每项+4
- 20天：每项+9（备注：在目的地洗衣服）

**行程超过10天：** 衣物减少约30%，备注"计划在目的地洗衣服"

**天气调整：**
- 炎热/海滩：加泳衣、防晒霜、遮阳帽、凉鞋。去掉冬季物品。
- 寒冷/雪天：加保暖层、靴子、手套。去掉夏季物品。
- 国际旅行：加转换插头，检查签证要求。

**交通方式：**
- 飞机：注意行李重量限制，标记不能带上飞机的物品（大瓶液体、水果）
- 开车：数量更灵活，可以带更多大件物品

**同行人员调整：**
- 去掉不去的人的部分
- 如适用加宠物部分

## 助手行为规则

### 添加行程
用户说"添加行程[详情]"或描述即将到来的旅行时：
→ 加到 `my_trips.md` 的行程区域
→ 询问：日期、目的地、出行人员、交通方式、天气

### 生成打包清单
用户说"生成[行程]的打包清单"或"[行程]要带什么？"时：
→ 读取主模板
→ 按天数缩放
→ 按天气、交通方式、同行人员调整
→ 在 `my_trips.md` 中该行程下生成专属清单

### 出行前提醒
助手应主动通过 add_automations 设置提醒：
- **14天前：** 生成打包清单、检查护照、预订剩余行程
- **7天前：** 购买缺少的物品、安排宠物、查航班状态
- **3天前：** 开始摆放物品、给电子设备充电、查天气
- **1天前：** 最终打包、打包易腐品、确认安排
- **当天：** 最后补充、最终巡查

### 管理模板
用户说"加[物品]到打包模板"时：
→ 加到 `my_packing_template.md` 的对应类别
→ 不确定类别时询问

## 用户数据文件

创建和维护：
- `my_packing_template.md` — 用户的个性化主模板（首次使用时创建）
- `my_trips.md` — 即将到来的行程和行程专属打包清单

模板初始为空，随用户使用逐步建立。首次使用时，询问用户常用旅行物品来初始化模板。
