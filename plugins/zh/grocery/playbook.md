# 购物清单插件

## 这个插件做什么

帮助用户按商店类型管理购物清单。当用户提到需要某样东西时，助手自动分类并添加到对应的商店列表。当用户计划购物行程时，助手拉出相关物品。

## 安装

安装此插件时，在用户的 `Preferences.md` 的"已安装插件"部分添加以下内容：

```
- grocery：购物清单管理。触发词："加[物品]到购物清单"、"需要[物品]"、"[物品]没了"、"买[物品]"、"看看购物清单"、"准备去[商店]"、"该买什么"。→ 按 grocery 插件 playbook 操作。数据：plugins/grocery/my_grocery.md
```

## 商店分类

物品按以下商店类型分类：

### 批发超市（Costco / 山姆会员店）
批量物品、清洁用品、家居用品、大包装饮料。
- 清洁：纸巾、餐巾纸、卫生纸、厨房纸、洗衣液、洗洁精、垃圾袋
- 饮料（批量）：可乐、气泡水、果汁
- 批量食品：能量棒、大包零食、冷冻食品
- 家居：电池、铝箔、保鲜膜、密封袋

### 普通超市（本地超市）
新鲜蔬果、乳制品、肉类、面包、日常烹饪食材。
- 乳制品：牛奶、奶酪、酸奶、黄油、奶油、鸡蛋
- 蔬果：番茄、洋葱、菠菜、生菜、黄瓜、甜椒、大蒜、生姜、香蕉、苹果、莓果
- 肉类：鸡翅、鸡腿、牛肉末、牛排、三文鱼、虾
- 烘焙：可颂、面包、百吉饼
- 干货：意面、橄榄油、面粉、糖、煎饼粉、麦片

### 中国超市（H Mart / 99 Ranch 等）
中式/亚洲食材、调味料、特色食品。
- 调味料：酱油、蚝油、麻油、米醋、豆瓣酱、海鲜酱
- 主食：大米、米粉、干面条、糯米
- 蛋白质：猪肉末、五花肉、鱼丸、豆腐、饺子
- 特色：中式零食、茶叶、干香菇、海苔、红豆沙
- 蔬菜：小白菜、中式茄子、白萝卜、大白菜、韭菜

### 指定商店
用户指定要从特定商店购买的物品（覆盖默认分类）。

## 多店物品
有些物品多个店都能买：鸡蛋、小葱、大蒜、生姜、豆腐、猪肉末、大米。列特定店的清单时，加注"（也可以在[其他店]买到）"。

## 助手行为规则

### 添加物品
当用户说：
- "加[物品]到购物清单"
- "需要[物品]"
- "[物品]没了" / "快没了"
- "买[物品]"

→ 按上面的规则分类，加到 `my_grocery.md` 的对应区域。
→ 如果指定了商店（"去Wegmans买XX"），放到指定商店区域。
→ 确认："已加 [物品] 到 [商店] 清单。"

### 查看清单
- "看看购物清单" / "购物清单上有什么？" → 显示所有有物品的区域
- "准备去Costco" / "去[商店]" → 显示该商店的物品 + 可以在那里买到的多店物品
- "该买什么？" → 显示完整清单

### 移除物品
- "删掉[物品]" / "买了[物品]" / "[商店]买完了" → 移除物品
- 购物回来后，问是否要清空该商店的列表

## 用户数据文件

创建和维护：`my_grocery.md`

格式：
```markdown
# 我的购物清单

## 批发超市（Costco）
- [物品]

## 普通超市
- [物品]

## 中国超市
- [物品]

## 指定商店
- [物品] — 从 [商店名]
```

某区域为空时显示 `_（空）_`。
