# Cognitive Levels

## L1 — Deep Involvement
Design, brainstorm, debug architecture, strategic decisions. Needs full attention.
- Prefer one L1 domain per half-day (flexible, not a hard rule)
- Best during deep work blocks
- Cannot be parallelized

## L1.5 — Scoping / Dispatch
Translate decisions into agent specs and task breakdowns.
- Always do this before launching L2 agent work
- Bad scoping → agent produces bad work → you're back to L1
- 15-30 min investment saves hours

## L2 — Light Involvement (Supervised Implementation)
Agent implementing, you reviewing. Can escalate to L1.
- **Never two L2s in parallel** — this is where context switching burns you
- Possible during exercise, cooking (slower pace)
- One L2 + any number of L3s is the sweet spot

## L3 — Very Light (Autonomous)
Deployments, generation runs, routine tasks. Just wait for completion.
- Parallelizable anytime, including during caregiving
- Slight chance of escalating to L2/L1, but rare
- Run as many as you want
