# 旅行打包清单

> **使用方法：** 这是多日旅行打包的总模板。出发前14天，agent 会根据此模板生成针对具体行程的打包清单，按照：行程天数、目的地天气、交通方式（开车/飞机）、出行人员等自动调整。Agent 还会提醒你出行前的准备任务。

## 出行前清单模板

> Agent 在出发前14天生成此清单。根据具体行程调整。

### 14天前
- [ ] 生成行程专属打包清单
- [ ] 检查所有出行人员护照有效期
- [ ] 预订剩余住宿/机票
- [ ] 查看目的地天气预报
- [ ] 确认哪些需要买、哪些已有

### 7天前
- [ ] 购买清单上缺少的物品
- [ ] 查看航班/车辆状态
- [ ] 洗一批旅行要穿的衣服
- [ ] 补充药品

### 3天前
- [ ] 开始在打包区域摆放物品
- [ ] 给所有电子设备充电（充电宝、平板等）
- [ ] 再次查看天气预报，调整衣物
- [ ] 打印登机牌/确认邮件

### 1天前
- [ ] 最终打包——对照清单检查
- [ ] 打包食物/饮料/零食（易腐品）
- [ ] 设置外出自动回复（如需要）
- [ ] 确认所有证件集中放置

### 出发当天
- [ ] 最后补充新鲜物品（水、零食）
- [ ] 最后巡查房间
- [ ] 再次确认证件、钱包、手机、钥匙

## 缩放规则（给 agent 用）

**衣物数量 = 基准 + ceil((行程天数 - 3) / 2)**
- 3天行程：基准数量
- 5天行程：每项+1
- 7天行程：每项+2
- 10天行程：每项+4
- 20天行程：每项+9（考虑在目的地洗衣服）

**行程超过10天：** 备注"计划在目的地洗衣服"，衣物减少约30%

**天气调整：**
- 炎热/海滩：加泳衣、防晒霜、遮阳帽、凉鞋。去掉冬季物品。
- 寒冷/雪天：加保暖层、靴子、手套、帽子。去掉夏季物品。
- 国际旅行：加转换插头。检查签证/证件要求。

**交通方式调整：**
- 飞机：注意行李重量限制。标记不能带上飞机的物品（大瓶液体）。必需品放随身行李。
- 开车：数量更灵活。可以带更多大件物品。
